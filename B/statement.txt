Problem Statement:
There are N chips packets numbered (1,2..N) containing A[i] chips in it.
Ketekon (King of Keteki Country), loves to eat a lot of chips,but, the king 
can take only k consequtive chips packets ,beacuse Keteki Country follows the K-Rule. 
Find the maximum number of chips which the king can have.

Input:
The first line of the input contains 2 space-separated integers N and K.
The second line contains N space separated integers denoting the number of chips in packet A[i].

Output:
Output the maximum number of chips, the King can have.

Constraints:
1<= N,K <=10^5
1<=K<=N
1<=A[i]<=10^9

Sample Input:
5 2
5 1 4 3 2
Sample Output:
7
